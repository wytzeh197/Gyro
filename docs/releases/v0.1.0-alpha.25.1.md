# Gyro v0.1.0-alpha.25.1

Gyro Alpha 25.1 is a focused desktop polish release that aligns the sidebar
controls and titlebar divider with the native macOS traffic-light geometry.

## Download and install

Gyro requires macOS 14+ and is not Apple-signed or notarized. The application
has a complete ad-hoc signature, so macOS can verify the bundle structure, but
the first launch still requires Apple's one-time **Open Anyway** approval.

Use the [Gyro download site](https://wytzeh197.github.io/Gyro/) to choose a
build, then follow the
[macOS installation guide](https://github.com/wytzeh197/Gyro/blob/main/docs/install-macos.md).
Do not disable Gatekeeper or remove quarantine globally.

- [Apple Silicon DMG](https://github.com/wytzeh197/Gyro/releases/download/v0.1.0-alpha.25.1/Gyro_0.1.0-alpha.25.1_aarch64.dmg)
- [Intel DMG](https://github.com/wytzeh197/Gyro/releases/download/v0.1.0-alpha.25.1/Gyro_0.1.0-alpha.25.1_x64.dmg)
- [SHA256SUMS](https://github.com/wytzeh197/Gyro/releases/download/v0.1.0-alpha.25.1/SHA256SUMS)

## What changed

- The sidebar hide, back, and forward controls now share the native traffic
  lights' vertical centerline while keeping their normal horizontal positions.
- The titlebar divider now sits at the correct macOS titlebar boundary.
- The transparent sidebar-restore affordance uses the same aligned geometry
  when the sidebar is hidden.
- Workbench smoke coverage now pins the Tauri traffic-light coordinates and the
  matching CSS titlebar geometry to prevent regressions.

## Behavior changes and known limits

- Gyro is still an unsigned public Alpha. A warning-free first launch requires
  Apple Developer ID signing and notarization, which this release does not use.
- Managed Macs may prohibit **Open Anyway**. Gyro does not bypass device policy.
- Apple Silicon and Intel remain separate downloads; there is no universal DMG.
- Homebrew installs the `gyro` CLI only. Gyro.app remains a direct DMG install.
- Alpha 25 sessions and settings remain compatible; no database or session
  schema migration is required.

## Upgrade and rollback

Alpha 25 sessions, attachments, plans, approvals, projects, and local provider
state remain compatible with Alpha 25.1. Update from Gyro or replace the
existing app with the matching Alpha 25.1 DMG.

To roll back, quit Gyro, back up
`~/Library/Application Support/Gyro/`, and reinstall
[`v0.1.0-alpha.25`](https://github.com/wytzeh197/Gyro/releases/tag/v0.1.0-alpha.25).

## CLI

This patch does not change CLI behavior. After the release Formula reaches the
tap:

```bash
brew tap wytzeh197/tap
brew trust --formula wytzeh197/tap/gyro
brew install gyro
```

Direct archives remain available for
[Apple Silicon](https://github.com/wytzeh197/Gyro/releases/download/v0.1.0-alpha.25.1/gyro-cli-0.1.0-alpha.25.1-aarch64-apple-darwin.tar.gz)
and
[Intel](https://github.com/wytzeh197/Gyro/releases/download/v0.1.0-alpha.25.1/gyro-cli-0.1.0-alpha.25.1-x86_64-apple-darwin.tar.gz).

## Alpha notice

This is a GitHub-built public macOS Alpha for Apple Silicon and Intel. Updater
archives are signed with Gyro's updater key, while app bundles use ad-hoc code
signing and DMGs are not Apple-signed or notarized.
