# Gyro v0.1.0-alpha.23.2

This follow-up release improves multi-chat work, queue steering, project
navigation, and live activity reporting in the Chat surface.

## What changed

- Active responses keep streaming in the background when switching chats or
  projects. Returning to a running chat restores the latest accumulated text
  without overwriting newer chunks with an older transcript snapshot.
- Queued follow-ups use a compact composer-attached row. A single trailing
  composer action switches between Stop and Queue/Send, while Steer stops the
  current response and promotes the selected queued message to run next.
- A queued message remains visible as Sending until its turn finishes. Waiting
  messages retain working remove and details controls.
- Sidebar projects keep their positions instead of jumping with recent
  activity. Project groups can be reordered by drag and drop or Alt+Arrow keys,
  and the chosen order persists across app restarts.
- The composer shows an accessible estimated context-usage wheel immediately
  before the model selector.
- Repeated file-edit activity is deduplicated by path into one live row per
  file. Addition and deletion totals refresh during active edits and can move
  up or down as the working tree changes.

## Behavior changes and known limits

- Context usage is an estimate derived from the visible thread because provider
  streams do not currently expose authoritative per-turn context consumption.
- Follow-up queues remain local to the current app run and are not restored
  after quitting or installing an update.
- Steer stops the current response and sends the promoted message next; it does
  not inject text into a provider process that is already running.
- No database or session-schema migration is required.

## Upgrade and rollback

Update from Gyro or replace the existing app with the new DMG. CLI users can
install the matching archive; a generated Formula is included as a release
asset for manual installation or publishing to a tap.

Existing sessions, attachments, approvals, and Alpha 23 mutation-recovery state
remain compatible with Alpha 23.2. To roll back, back up
`~/Library/Application Support/Gyro/` and reinstall `v0.1.0-alpha.23.1`.

## Alpha notice

This is a GitHub-built public macOS alpha for Apple Silicon and Intel. Updater
archives are signed, but app bundles and DMGs are not Apple-signed or notarized.
