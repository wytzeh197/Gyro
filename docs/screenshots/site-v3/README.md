# Website product visuals

These are the full-resolution sources for the current Gyro public website.

- `hero-1920x1080.png` — the user-provided visual direction and homepage hero.
- `chat-1672x941.png` — a completed bounded-retry conversation.
- `cli-1672x941.png` — local Git and test output in two populated panes.
- `workspace-1672x941.png` — changed files, diff, tests, and review controls.
- `social-preview-1200x630.png` — sharing crop derived from the hero.

The Chat, CLI, and Workspace visuals were generated from the hero as a style
reference. They use fictional demo content and contain no private repository,
account, or provider data. Optimized WebP derivatives live under
`site/assets/screenshots/`.
