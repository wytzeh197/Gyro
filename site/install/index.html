First launch

Open Gyro safely.

Three steps. You only approve the app once.

  1. 01

    Move Gyro to Applications.

    Open the DMG and drag Gyro into Applications.

  2. 02

    Try to open Gyro.

    Dismiss the macOS security warning.

  3. 03

    Choose Open Anyway.

    Open System Settings → Privacy & Security, select Open Anyway, then confirm Open.

Managed Mac?

Your administrator may block Open Anyway. Gyro cannot override that policy.

Help and recovery.

Updates and rollback

Gyro verifies its own updater signatures. Apple signing is separate. To roll back, download a previous DMG, quit Gyro, replace the app in Applications, and reopen it.

Browse the changelog
Uninstall Gyro

Quit Gyro and move Gyro.app from Applications to the Trash. To remove local sessions and settings too, delete ~/Library/Application Support/Gyro.

Install the CLI with Homebrew 
brew tap wytzeh197/tap
brew trust --formula wytzeh197/tap/gyro
brew install gyro

The tap installs the CLI only, not Gyro.app.

Build from source 
git clone https://github.com/wytzeh197/Gyro.git
cd Gyro
pnpm install
pnpm desktop:build

Troubleshooting and support

If the DMG will not open, confirm macOS 14+, the correct processor build, and the published SHA-256. Do not use terminal quarantine bypasses.
