Unsigned public alpha

Gyro is not Apple-signed or notarized. Approve the first launch in Privacy & Security. Follow the safe install steps—never disable Gatekeeper or remove quarantine.

First launch

Install without lowering your guard.

Approve Gyro once with Apple’s supported flow.

  1. 01

    Download the correct DMG.

    Choose Apple Silicon or Intel.

  2. 02

    Open it and drag Gyro to Applications.

    Open the installed copy.

  3. 03

    Attempt to open Gyro.

    Dismiss the unsigned-app warning.

  4. 04

    Open System Settings → Privacy & Security.

    Find Gyro under Security.

  5. 05

    Select Open Anyway, then confirm Open.

    Authenticate if asked.

Using a managed Mac? Open Anyway may be blocked. Gyro cannot override an administrator’s security policy.

Verify the download

Integrity, in the open.

Compare GitHub’s SHA-256 before opening the DMG.

Intel DMGLoading SHA-256…

In Terminal, run shasum -a 256 Gyro_VERSION_ARCH.dmg and compare the result above or in SHA256SUMS.

Uninstall Gyro +

Quit Gyro and move Gyro.app from Applications to the Trash. Your local sessions remain in ~/Library/Application Support/Gyro/. Back that folder up before deleting it if you want to remove local data too.

Roll back to an earlier alpha +

Back up ~/Library/Application Support/Gyro/, download the needed version from previous releases, and replace Gyro.app. Read that release’s compatibility notes before reopening your sessions.

Build from source +

Install Node.js 22+, pnpm 11+, Rust 1.78+, and Xcode Command Line Tools, then run: 

git clone https://github.com/wytzeh197/Gyro.git
cd Gyro
corepack enable
pnpm install --frozen-lockfile
pnpm doctor
pnpm desktop:install-local

Read the complete source-build guide.
