First launch

Open Gyro safely.

Approve the unsigned alpha once with Apple’s supported flow.

  1. 01

    Move Gyro to Applications.

    Open the DMG and drag Gyro into Applications.

  2. 02

    Try to open Gyro once.

    Dismiss the unsigned-app warning.

  3. 03

    Choose Open Anyway.

    Open System Settings → Privacy & Security, then confirm Open.

Using a managed Mac? Open Anyway may be blocked. Gyro cannot override an administrator’s security policy. Apple’s security guidance

Uninstall or roll back

To uninstall, quit Gyro and move Gyro.app from Applications to the Trash. Local sessions remain in ~/Library/Application Support/Gyro/.

To roll back, back up that folder, download an earlier alpha from previous releases, and replace Gyro.app.

Build from source

Install Node.js 22+, pnpm 11+, Rust 1.78+, and Xcode Command Line Tools, then run: 

git clone https://github.com/wytzeh197/Gyro.git
cd Gyro
corepack enable
pnpm install --frozen-lockfile
pnpm doctor
pnpm desktop:install-local
