First launch

Install without lowering your guard.

Apple allows you to approve a known app once. These are the supported steps for Gyro’s unsigned alpha.

  1. 01

    Download the correct DMG.

    Choose Apple Silicon for M-series Macs or Intel for an Intel Mac.

  2. 02

    Open it and drag Gyro to Applications.

    Launch the installed copy from your Applications folder.

  3. 03

    Attempt to open Gyro.

    macOS will block this first attempt because the alpha is not notarized. Dismiss the warning.

  4. 04

    Open System Settings → Privacy & Security.

    Scroll to Security, where macOS records the blocked Gyro launch.

  5. 05

    Select Open Anyway, then confirm Open.

    Authenticate if macOS asks. Future launches of that copy should open normally.

Using a managed Mac? Your organization may prohibit the Open Anyway exception. Gyro cannot and will not override an administrator’s security policy.

Verify the download

Integrity, in the open.

GitHub publishes a SHA-256 digest for every asset. Compare it before opening a download if you want an additional integrity check.

In Terminal, run shasum -a 256 Gyro_VERSION_ARCH.dmg and compare the result above or in SHA256SUMS.

Uninstall Gyro +

Quit Gyro and move Gyro.app from Applications to the Trash. Your local sessions remain in ~/Library/Application Support/Gyro/. Back that folder up before deleting it if you want to remove local data too.

Roll back to an earlier alpha +

Back up ~/Library/Application Support/Gyro/, download the needed version from previous releases, and replace Gyro.app. Read that release’s compatibility notes before reopening your sessions.

Build from source +

Install Node.js 22+, pnpm 11+, Rust 1.78+, and Xcode Command Line Tools, then run: 

git clone https://github.com/wytzeh197/Gyro.git
cd Gyro
corepack enable
pnpm install --frozen-lockfile
pnpm doctor
pnpm desktop:install-local
